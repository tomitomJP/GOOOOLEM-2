作品名

MAL

概要

本作は、戦車を操作し、緑色の球体あるMALを倒しポイントを集めるゲームです。

ゲーム内容

ジャンル：見下ろし型ローグライクシューティング

プレイ人数：1人

プレイ時間目安：1プレイ 約10-30分

対応プラットフォーム：PC（Windows）

特徴・工夫点

インベントリを操作して発動する効果を任意に選択できるシステム

ランダム生成によるリプレイ性

操作の分かりやすさを重視したUI設計

操作方法
操作	内容
WASD / スティック	移動
マウス / 右スティック	チャージ攻撃
クリック / ボタン	攻撃
TAB / インベントリ開閉
使用技術

エンジン：Unity 2022 LTS

言語：C#

その他：Input System / ScriptableObject など

実行方法
Unityプロジェクトの場合

本リポジトリをクローン

Unity Hubでプロジェクトを開く

MainScene を再生

ビルド版がある場合

Build フォルダ内の実行ファイルを起動してください
